เครื่องคำนวณการสูญเสียสมรรถภาพ — โรคหลอดเลือดหัวใจ

เลือกขั้นความรุนแรง (ตาราง 10-4) แล้วเลือกระดับย่อย A–E → ร้อยละการสูญเสียทั้งร่างกาย (WPI) · คำนวณล้วน ไม่บันทึกข้อมูล

⚠ ข้อควรทราบ
  • ประเมินเมื่ออาการคงที่ ได้รับการรักษาอย่างเหมาะสมเต็มที่แล้ว (maximum medical improvement)
  • ขั้นความรุนแรงพิจารณาจาก ประวัติ + การตรวจร่างกาย + การฉีดสีหลอดเลือด (angiography) ประกอบกัน
  • ระดับย่อย A–E ภายในขั้น ใช้ดุลยพินิจตามความรุนแรงจริงของผู้ป่วยในช่วงร้อยละของขั้นนั้น
  • โมดูลรุ่นนี้ครอบคลุม “โรคหลอดเลือดหัวใจ (coronary heart disease)” — ความดันโลหิตสูง หลอดเลือดส่วนปลาย ลิ้นหัวใจ และหัวใจล้มเหลว จะเพิ่มในรอบถัดไป
  • การตัดสินขั้นสุดท้ายอยู่ที่ดุลยพินิจของแพทย์ผู้ประเมิน

ระดับความสามารถของหัวใจ (NYHA functional class)

อ้างอิงประกอบการจัดขั้น (ตาราง 10-1)
ดูเกณฑ์ NYHA class I–IV

ขั้นความรุนแรงของโรคหลอดเลือดหัวใจ (ตาราง 10-4)

แตะเลือกขั้นที่ตรงกับผู้ป่วย · แสดงเกณฑ์ประวัติ / การตรวจ / การฉีดสี

ระดับย่อยภายในขั้น (severity grade A–E)

A = น้อยที่สุดในขั้น · E = มากที่สุดในขั้น (ค่าตรงตามตาราง 10-4)

ผลการประเมิน

การตัดสินขั้นสุดท้ายอยู่ที่ดุลยพินิจของแพทย์ผู้ประเมิน
