แบบฟอร์ม รสส. (ผลตรวจวัดสภาพแวดล้อม)

ค่ามาตรฐานอ้างอิงกฎกระทรวงกำหนดมาตรฐานฯ ความร้อน แสงสว่าง เสียง พ.ศ. 2559 · ข้อมูลนี้จะแสดงในรายงาน (สร้างรายงาน) เป็นหัวข้อ “ผลการตรวจวัดสภาพแวดล้อมในการทำงาน”

การทำงานร่วมกันแบบเรียลไทม์

ทีมสำรวจเข้า ห้องเดียวกัน ด้วยรหัสสำรวจ + รหัสผ่านทีม แล้วแก้ข้อมูลพร้อมกัน เห็นการเปลี่ยนแปลงสดทันที
สร้างห้องใหม่จะอัปโหลดข้อมูลปัจจุบันขึ้นเป็นข้อมูลเริ่มต้นของห้อง · เข้าร่วมห้องจะดึงข้อมูลล่าสุดจากทีมมาแทนที่เครื่องนี้
ใส่ค่าโปรเจกต์ Supabase ของคุณ (Project URL + anon public key) — เก็บไว้ในเครื่องนี้เท่านั้น

1
เปิด Supabase → เมนู SQL Editor → New query
2
กด คัดลอก SQL ด้านล่าง แล้ววางแล้วกด Run — ระบบจะสร้างตาราง + ฟังก์ชันแบบเข้ารหัส (anon key ดึงข้อมูลตรง ๆ ไม่ได้ ต้องมีรหัสผ่านห้อง — สอดคล้อง PDPA)
